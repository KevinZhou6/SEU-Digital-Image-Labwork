## 配置MinGW+Opencv-4.5.2 

<a href= "https://github.com/huihut/OpenCV-MinGW-Build"> Opencv下载

1. 下载文件之后，先将opencv的压缩文件，解压到qt安装目录的同级文件路径下
2. 添加qt的mingw的bin，opencv对应的bin文件到系统环境变量中
3. 在本地选择mingw64编译器编译工程即可
